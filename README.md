# ⚡ ARROW SURGE

> A neon arcade survival game — navigate your ship through endless walls and survive as long as possible.

**Live Demo:**  

---

## 🎮 Gameplay

Control a glowing diamond ship using arrow keys or the on-screen D-pad.  
Walls scroll toward you — find the gap and pass through before it's too late.  
Each crash costs a life. Lose all 3 and it's game over.

- ⬆️⬇️⬅️➡️ Move in all 4 directions
- 🧱 Walls have a gap — navigate through it
- ❤️ 3 lives total
- ⚡ Speed and difficulty increase over time
- 🏆 Best score saved in your browser

## 🛠️ Built With

- **HTML5 Canvas** — all game rendering
- **Vanilla JavaScript** — game logic, no frameworks
- **CSS3** — glassmorphism UI, animations
- **Google Fonts** — Orbitron + Rajdhani

No backend. No dependencies. No build step.
