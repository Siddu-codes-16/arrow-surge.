## 🎮 Gameplay

Control a glowing diamond ship using arrow keys or the on-screen D-pad.  
Walls scroll toward you — find the gap and pass through before it's too late.  
Each crash costs a life. Lose all 3 and it's game over.

- ⬆️⬇️⬅️➡️ Move in all 4 directions
- 🧱 Walls have a gap — navigate through it
- ❤️ 3 lives total
- ⚡ Speed and difficulty increase over time
- 🏆 Best score saved in your browser

## 🚀 Deploy to Vercel

### 1. Fork or Clone this repo

```bash
git clone https://github.com/YOUR_USERNAME/arrow-surge.git
cd arrow-surge
```

### 2. Push to GitHub

```bash
git add .
git commit -m "Initial commit"
git push origin main
```

### 3. Deploy on Vercel

1. Go to [vercel.com](https://vercel.com) and sign in with GitHub
2. Click **"Add New Project"**
3. Import this repository
4. Set **Framework Preset** → `Other`
5. Leave Build Command and Output Directory **empty**
6. Click **Deploy** ✅

Your game goes live in about 1 minute.
